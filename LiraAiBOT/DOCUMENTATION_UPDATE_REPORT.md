## 🚀 Команды для публикации

```bash
cd /Users/iluyshin.d/Desktop/LiraAiBOT

# 1. Инициализация
git init

# 2. Добавление всех файлов
git add .

# 3. Первый коммит
git commit -m "feat: Full documentation update for GitHub

- Updated README with payment system info
- Added CONTRIBUTING, CHANGELOG, SECURITY
- Added GitHub Issue and PR templates
- Created GITHUB_SETUP guide
- Updated .gitignore for production"

# 4. Создание ветки main
git branch -M main

# 5. Добавление remote (замените YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/LiraAiBOT.git

# 6. Отправка на GitHub
git push -u origin main
```
