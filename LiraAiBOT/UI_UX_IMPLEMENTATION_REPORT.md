## 🎯 Интеграция

### Команда `/stats`

**Было**:
```python
stats_text = f"""📊 **Ваша статистика**

👤 {name}
🔑 Уровень: **{level_info.get(level, 'Пользователь')}**

📈 Генерации изображений:
• Сегодня: **{daily_count}/{daily_limit}**
• Всего: {total_count}
...
"""
```

**Стало**:
```python
from backend.utils.formatters import format_stats_card

stats['daily_limit'] = limit_info.get('daily_limit', 3)
stats_text = format_stats_card(stats)
```

### Callback "stats"

**Было**:
```python
stats_text = f"""📊 **Статистика**

• Сегодня: {daily_count}
• Всего: {total_count}

```

**Стало**:
```python
from backend.utils.formatters import format_short_stats

stats['daily_limit'] = limit_info.get('daily_limit', 3)

```

## 🎨 Дизайн-система

### Эмодзи для уровней

| Уровень | Эмодзи | Описание |
|---------|--------|----------|
| `admin` | 👑 | Админ (безлимит) |
| `sub+` | 🚀 | sub+ (30/день) |
| `subscriber` | ⭐ | Подписчик (5/день) |
| `user` | 👤 | Пользователь (3/день) |

### Статусы лимитов

| Статус | Эмодзи | Условие |
|--------|--------|---------|
| Доступно | ✅ | < 80% лимита |
| Скоро лимит | ⚠️ | ≥ 80% лимита |
| Лимит исчерпан | 🔒 | 100% лимита |
| Безлимит | ∞ | limit ≤ 0 или -1 |

### Прогресс-бар

```
██████████░░░░░░░░░░ 10/20 (50%)
```

- `█` - заполненная часть
- `░` - пустая часть
- Длина: 20 символов
- Формат: `{bar} {current}/{total} ({percentage}%)`

## 📝 Примечания

### Технические детали

1. **Кэширование**: `get_user_stats` читает напрямую из Supabase (кэш отключён)
2. **TIMEZONE**: Конвертация UTC → Moscow time реализована
3. **Fallback**: SQLite поддерживается для локальной разработки

### Известные ограничения

1. **Длина сообщения**: Карточка занимает ~25 строк (лимит Telegram 4096 символов)
2. **Моноширинный шрифт**: Рамка может "плыть" на разных устройствах
3. **Эмодзи**: Некоторые эмодзи могут отображаться по-разному
